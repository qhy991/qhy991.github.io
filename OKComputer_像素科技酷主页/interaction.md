# 交互设计文档

## 核心交互功能

### 1. 像素风终端界面
- **功能描述**: 创建一个复古终端风格的命令行界面，用户可以输入命令查看不同内容
- **交互方式**: 
  - 用户可以在终端中输入命令如 `about`, `projects`, `publications`, `contact`
  - 终端会显示相应的信息，带有打字机效果
  - 支持历史命令浏览（上下箭头）
  - 自动补全功能（Tab键）

### 2. 研究兴趣雷达图
- **功能描述**: 交互式的技能雷达图，展示不同研究领域的能力水平
- **交互方式**:
  - 鼠标悬停显示详细描述
  - 点击可以展开相关项目
  - 动态数据可视化，使用ECharts.js
  - 支持切换不同视图（学术、技术、工具）

### 3. 项目时间线浏览器
- **功能描述**: 交互式时间线，展示学术历程和项目发展
- **交互方式**:
  - 水平滚动浏览不同时间段
  - 点击时间节点查看详细信息
  - 项目卡片翻转动画
  - 筛选功能（按类型、年份、技术栈）

### 4. 学术成就展示板
- **功能描述**: 游戏化的成就系统，展示学术里程碑
- **交互方式**:
  - 像素风格的成就徽章
  - 鼠标悬停显示解锁条件和时间
  - 进度条显示当前目标
  - 解锁动画效果
