# 设计风格文档

## 设计理念

### 核心概念：像素未来主义
融合复古像素美学与现代科技感，创造独特的视觉语言。通过8位风格的界面元素和现代化的交互设计，体现从经典到未来的技术演进。

### 颜色方案
- **主色调**: 深蓝色 (#0f172a) - 科技感和专业性
- **强调色**: 青色 (#06b6d4) - 现代感和活力
- **辅助色**: 紫色 (#8b5cf6) - 创新和神秘
- **高亮色**: 绿色 (#10b981) - 成功和进步
- **中性色**: 灰色 (#64748b) - 平衡和稳定

### 排版设计
- **标题字体**: "JetBrains Mono" - 等宽字体，体现编程文化
- **正文字体**: "Inter" - 现代无衬线字体，易读性强
- **像素字体**: "Press Start 2P" - 8位风格，用于装饰元素

### 视觉语言
- **几何形状**: 方形和矩形为主，体现像素网格
- **边框样式**: 1-2px实线，营造像素感
- **阴影效果**: 投影和发光，增强层次感
- **渐变**: 微妙的线性渐变，避免过度饱和

## 视觉效果

### 背景效果
- **动态网格**: 使用p5.js创建流动的像素网格背景
- **粒子系统**: 浮动的光点，模拟数字空间
- **渐变叠加**: 从深蓝到紫色的径向渐变

### 动画效果
- **打字机效果**: 文字逐字符显示
- **像素化过渡**: 元素出现时从像素块组装
- **发光脉冲**: 重要元素的呼吸灯效果
- **数据流**: 模拟数据传输的视觉效果

### 交互反馈
- **悬停发光**: 鼠标悬停时元素发光
- **点击波纹**: 点击时的扩散效果
- **状态变化**: 平滑的颜色和形状过渡
- **加载动画**: 像素风格的进度指示器

## 布局设计

### 网格系统
- **基础网格**: 12列响应式网格
- **像素对齐**: 所有元素严格对齐到8px网格
- **间距规范**: 8px、16px、24px、32px的倍数

### 组件设计
- **按钮**: 圆角矩形，像素风格边框
- **卡片**: 悬浮效果，微妙的阴影
- **导航**: 顶部固定，半透明背景
- **模态框**: 中心弹出，背景模糊

### 响应式断点
- **桌面**: 1200px+
- **平板**: 768px - 1199px
- **手机**: 320px - 767px

## 图标设计

### 风格特征
- **像素完美**: 16x16或32x32网格
- **简洁线条**: 2px线宽
- **统一风格**: 一致的视觉语言
- **功能明确**: 直观易懂

### 图标类型
- **技术图标**: 芯片、电路、代码等
- **社交图标**: GitHub、邮箱、学术平台
- **功能图标**: 下载、分享、收藏等
- **装饰图标**: 星星、箭头等

## 图像处理

### 头像设计
- **像素风格**: 8位风格的个人头像
- **发光效果**: 边缘发光，突出显示
- **动画**: 轻微的浮动效果

### 项目图片
- **截图风格**: 保持原始界面风格
- **统一尺寸**: 16:9比例
- **优化处理**: 压缩和格式优化

## 用户体验

### 可访问性
- **颜色对比**: 确保4.5:1的对比度
- **键盘导航**: 支持Tab键导航
- **屏幕阅读器**: 语义化HTML结构
- **动画控制**: 提供减少动画的选项

### 性能优化
- **图片懒加载**: 提升初始加载速度
- **CSS优化**: 使用transform和opacity
- **JavaScript优化**: 防抖和节流
- **缓存策略**: 合理设置缓存头

## 品牌一致性

### 视觉识别
- **Logo设计**: 简化的像素风格标识
- **色彩应用**: 严格遵循配色方案
- **字体使用**: 统一的字体层级
- **间距规范**: 一致的留白处理

### 内容语调
- **专业严谨**: 学术内容的准确性
- **现代感**: 技术前沿的形象
